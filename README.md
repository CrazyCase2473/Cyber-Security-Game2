# Bug-Bash-Binary: Cybersecurity Defense

## Project Overview
**Bug-Bash-Binary** is a narrative-driven defense game developed in Unity. Players take on the role of Beau Bockus, a new employee at Bockus Accounting, who must defend the company's infrastructure during a massive, undisclosed cyberattack.

## Gameplay Mechanics
* **Bug-Bash-Binary Suite:** A visualization system that represents malicious code as interactive "bugs." Players must neutralize these threats via cursor interaction.
* **Security Decision Points:** To simulate the pressure of a real-world breach, players must answer cybersecurity-related questions to maintain system integrity. 
* **Failure States:** * **System Overload:** Failing to clear bugs leads to a full-scale security breach.
    * **Administrative Failure:** Incorrect responses to security questions result in immediate termination.

## Development Reflection
This project highlights a pivot toward a more focused UI-centric design. By leveraging Unity’s UI systems and a streamlined "point-and-clear" mechanic, we created an experience that effectively communicates the high-pressure environment of corporate cybersecurity.
